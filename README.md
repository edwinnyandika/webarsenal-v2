# 🕷️ WebArsenal\n\n**The Ultimate Web Scraping & Mirroring Toolkit (110 Scripts)**\n\n![Version](https://img.shields.io/badge/version-v4.0.0-blue.svg)\n![License](https://img.shields.io/badge/license-MIT-green.svg)\n\n> The definitive, battle-hardened collection of exactly 110 advanced modules for cloning sites, bypassing auth, extracting granular data, pushing to cloud vectors, and orchestrating full-scale monitoring operations.\n\n## Modules Framework\n\n### 1. `core/` (Heavy-Duty Mirrors)\nHigh-performance monolithic scripts to recursively download entire sites.\n- To execute: `node core/super-mirror.js --url https://example.com --depth 4`\n- To execute (Playwright): `node core/grab-playwright.js --url https://example.com --pdf`\n\n### 2. `scrapers/` (Targeted Data Extraction)\nScripts for granular operations.\n- **SPA React/Vue scraper**: `node scrapers/spa-scraper.js --url https://example.com --wait-for '#app'`\n- **E-commerce prices**: `node scrapers/ecommerce-scraper.js --url https://store.com`\n\n### 3. `analyzers/` (DOM & SEO Security)\nHeavy DOM tree analysis tools.\n- **SEO Audit**: `node analyzers/seo-auditor.js --url <URL>`\n- **Unused CSS**: `node analyzers/unused-css.js --url <URL>`\n\n### 4. `auth-helpers/` (Bypasses & Tokens)\nGet past walls.\n- **Cloudflare Bypass**: `node auth-helpers/cf-clearance-puller.js --url <URL>`\n- **Cookie Cloner**: `node auth-helpers/cookie-cloner.js --profile C:\Chrome\Profile`\n\n### 5. `exporters/` (Data Shifting & Translation)\nShift formats effortlessly.\n- **JSON to SQLite**: `node exporters/to-sqlite.js --input data.json`\n- **Full Archive WARC**: `node exporters/to-warc.js --dir ./mirrored`\n\n### 6. `integrations/` (Cloud Hooks)\nPush straight to the database or cloud.\n- **Airtable Sink**: `node integrations/airtable-sync.js --input records.json --token XYZ`\n- **AWS S3 Push**: `node integrations/aws-s3-uploader.js --dir site_data/ --bucket my-scrape-bucket`\n\n### 7. `monitors/` (Jobs & Tracking)\nPersistent jobs for monitoring.\n- **Change Detector**: `node monitors/change-detector.js --url <URL> --xpath "//div[@class='price']"`\n- **Discord Webhook**: `node monitors/discord-webhook.js --webhook <URL>`\n\n### 8. `utils/` (Core Tools)\nInfrastructure mechanics.\n- **Proxy Rotator**: `node utils/proxy-rotator.js --list proxies.txt`\n\n## Quickstart Playbook: Combined Orchestration\nTo execute complex workflows, chain these scripts together sequentially. For example, a scheduled Cloudflare-protected E-commerce scrape to S3:\n\n```bash\n1. node auth-helpers/cf-clearance-puller.js --url https://store.com\n2. node core/super-mirror.js --url https://store.com --depth 3 --cookie-jar cf_cookies.json\n3. node integrations/aws-s3-uploader.js --dir ./super-mirrored-site --bucket website-backups\n4. node integrations/slack-alerter.js --webhook https://hooks.slack.com/services/XYZ --message "Backup Success"\n```
