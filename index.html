Getting Started

WebArsenal requires Node.js v18+. No global installs needed — all 110 modules run directly via node.

Installation

git clone https://github.com/edwinnyandika/webarsenal.git
cd webarsenal
npm install

Verify

node core/super-mirror.js --help
All flags are documented via --help on every module. No man pages or docs required.

Core Modules

Heavy-duty recursive cloners. Download an entire deployed site including assets, CSS, JS, images, fonts.

# Full mirror with depth 4
node core/super-mirror.js --url https://example.com --depth 4

# Playwright-powered mirror (for SPA, CSR sites)
node core/grab-playwright.js --url https://example.com --pdf

Scrapers

Granular, targeted extraction. Use these for structured data, not full-site cloning.

# SPA scraper — waits for element before extracting
node scrapers/spa-scraper.js --url https://app.com --wait-for '#app'

# E-commerce price scraper
node scrapers/ecommerce-scraper.js --url https://store.com

Analyzers

# Full SEO audit with score
node analyzers/seo-auditor.js --url https://mysite.com

# Detect and report unused CSS
node analyzers/unused-css.js --url https://mysite.com

Auth Helpers

Only use on systems you own or have explicit written permission to test.
# Pull Cloudflare clearance token
node auth-helpers/cf-clearance-puller.js --url https://protected.com

# Clone cookies from a Chrome profile
node auth-helpers/cookie-cloner.js --profile "C:\Users\You\AppData\Local\Google\Chrome\Default"

Pipeline Orchestration

Chain modules together for complete extraction pipelines. This example bypasses Cloudflare, mirrors a protected site, uploads to S3, and alerts Slack.

# Step 1: Bypass Cloudflare
node auth-helpers/cf-clearance-puller.js --url https://store.com

# Step 2: Deep mirror with cookie auth
node core/super-mirror.js --url https://store.com --depth 3 --cookie-jar cf_cookies.json

# Step 3: Upload to S3
node integrations/aws-s3-uploader.js --dir ./super-mirrored-site --bucket website-backups

# Step 4: Alert team on Slack
node integrations/slack-alerter.js --webhook YOUR_WEBHOOK --message "Mirror complete ✓"

About the Project

Built for devs
who move fast.

WebArsenal is the toolkit the web developer community needed but didn't have — 110 focused, composable modules covering every phase of web data extraction.

The problem
it solves

Web extraction is fragmented. Developers stitch together Puppeteer snippets, Cheerio tutorials, half-finished proxy scripts, and manual exports just to get data out of a site they own.

There was no single, well-organized toolkit that handles the full pipeline — from "clone this site" through to "push results to Notion." WebArsenal solves that.

  • Self-contained scripts — one job per file, no hidden state
  • CLI-first — all flags, stdout output, pipe anywhere
  • Composable — output of one is natural input to the next
  • Zero SaaS — no API keys, no dashboards, no subscriptions
  • MIT licensed — use in commercial products freely

Who it's for

Full-stack devs, SEO engineers, data engineers, indie hackers, and security researchers (on systems they're authorized to test) all have something in this toolkit.

It's the kind of repo you bookmark and actually use — not a framework to learn, just scripts that work.
